## ✨ Features

| | Feature | Description |
|---|---------|-------------|
| 🔴 | **One-Click Disable** | Instantly hides and locks the Windows TaskBar |
| 🟢 | **One-Click Enable** | Restores the TaskBar to its original state |
| 🔔 | **System Tray** | Runs silently in the background after closing |
| 🚀 | **Auto-Start** | Optional launch at Windows startup |
| 💾 | **Boot Persistence** | Remembers your last setting across reboots |
| 🔒 | **Single Instance** | Prevents multiple copies from running |
| 🌙 | **Dark UI** | Clean, modern interface with smooth hover effects |
| 📦 | **Portable** | Zero installation, single `.exe` file |
| 🛡️ | **Safe Exit** | Always restores the TaskBar before closing |

## 🛠️ Usage
1️⃣ Download the .exe from Releases

2️⃣ Run TaskBarHider.exe

3️⃣ Click "Disable" to hide the TaskBar

4️⃣ Click "Enable" to bring it back

5️⃣ Close the window — it stays in the system tray! 🔔

> 💡 **Tip:** Enable **Auto-Start** to have it launch automatically with Windows!
