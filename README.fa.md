# مخفی‌کننده تسک‌ بار (Taskbar Hider)

> 🎓 این اولین پروژه منه! پیشنهادات و نظرات شما باعث خوشحالی من می‌شه 💛

یک برنامه ساده و تمیز برای مخفی و نمایش تسک‌بار ویندوز فقط با یک کلیک

![Platform](https://img.shields.io/badge/platform-Windows-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Version](https://img.shields.io/badge/version-1.0.0-orange)

🌐 **زبان‌های دیگر:** [English](README.md)

## ✨ ویژگی‌ها

- 🎯 رابط کاربری ساده و مینیمال
- 👁️ مخفی و نمایش فوری تسک‌بار ویندوز
- 🚀 قابلیت اجرا در استارت‌آپ ویندوز
- 💾 سبک و پرتابل (بدون نیاز به نصب)
- 🆓 کاملاً رایگان و متن‌باز

## 📥 دانلود

آخرین نسخه رو می‌تونید از بخش [Releases](../../releases) دانلود کنید

## 🚀 نحوه استفاده

۱. فایل `TaskBarHider.exe` رو از بخش Releases دانلود کنید
۲. برنامه رو اجرا کنید
۳. با کلیک روی دکمه، تسک‌بار رو مخفی یا نمایش بدید
۴. اگر می‌خواید برنامه با ویندوز اجرا بشه، گزینه **«اجرا با ویندوز»** رو فعال کنید
