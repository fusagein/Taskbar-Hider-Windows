## 🛠️ نحوه استفاده

1️⃣ فایل `.exe` را از بخش Releases دانلود کنید

2️⃣ فایل `TaskBarHider.exe` را اجرا کنید

3️⃣ روی گزینه `Disable` کلیک کنید تا تسک‌بار مخفی شود

4️⃣ روی گزینه `Enable` کلیک کنید تا تسک‌بار برگردد

5️⃣ با بستن پنجره، برنامه در System Tray فعال می‌ماند 🔔

> 💡 **نکته:** گزینه **Auto-Start** را فعال کنید تا برنامه همراه با ویندوز اجرا شود
